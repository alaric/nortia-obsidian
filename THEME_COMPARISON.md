## Key Differences

### Night vs Dusk
- **Background**: Night is 8% darker (more contrast with text)
- **Green backgrounds**: Dusk has richer, more saturated greens
- **Overall feel**: Night = deeper, Dusk = warmer

### Dawn vs Day
- **Background**: Day is 4% brighter (near-white vs off-white)
- **Palette 5** (cyan): Dawn uses #0576FF, Day uses #24A0FF (slightly lighter)
- **Overall feel**: Dawn = soft/cozy, Day = crisp/clear

## Choosing Your Theme

**Pick Night if you:**
- Work in completely dark rooms
- Prefer maximum contrast
- Work late into the night
- Are sensitive to bright screens

**Pick Dusk if you:**
- Have some ambient lighting
- Prefer warmer dark themes
- Work in the evening hours
- Want less stark contrast than Night

**Pick Dawn if you:**
- Work with natural morning light
- Prefer softer light themes
- Find pure white backgrounds too harsh
- Like a gentle, cozy atmosphere

**Pick Day if you:**
- Work in bright environments
- Need maximum clarity and sharpness
- Have lots of natural daylight
- Prefer crisp, high-contrast light themes

## Testing Recommendations

Try each theme in its intended environment:
1. **Morning (7-11am)**: Start with Dawn
2. **Afternoon (12-4pm)**: Switch to Day
3. **Evening (5-8pm)**: Try Dusk
4. **Night (9pm-6am)**: Use Night

After a few days, you'll know which feels most comfortable for each time of day!
